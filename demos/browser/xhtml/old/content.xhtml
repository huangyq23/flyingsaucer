Test 14: How do I make an XPATH node query?

If you want to select a set of nodes from an XML document you should start with the top node and select on down.

ex: if you have this XML

<root>

  <sub1>

    <sub2>

      <target>the answer</target>

    </sub2>

  </sub1>

</root>

then this XPATH

/root/sub1/sub2/target

will return this text

the answer
