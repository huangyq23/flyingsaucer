Pure CSS Layouts

2-column Pure-CSS Fluid Layout with Floating Menu: This sample page, using ideas from Intensiv Station Templates at http://www.intensivstation.ch, shows how to create a "two-column" layout with a floating menu, using just CSS. This is a nice 2-column fluid layout; the trick is using a background color on the content to give a visual cue that it's a separate section, and then floating it left with a % size. The "menu" on the right is actually just the text that is not in the main content div, placed there because the content is floated. The menu has no separate block or size, because the background color makes it look like a separate space, differentiated from the content block. Nice, simple trick.

As you adjust the size of the containing panel (say, the browser), the page will "reflow" to accomodate. Note in this example the layout will break because of "pre" block below will not overflow nicely--but holds up pretty well till you reduce beyond the minimum necessary for the preserved formatting.

A limitation with this layout is that we can't force the "content" section to be a minimum size (like % of viewport). It will grow to the size necessary to contain the content you give it.

        body {
        background-color: #e1ddd9;
        font-size: 12px;
        font-family: Verdana;
        color:#564b47;
        }

        #content {
        float: left;
        width: 70%;
        background: #fff;
        border-right: 2px solid #996666;
        border-bottom: 2px solid #996666;
        margin-right: 15px;

        padding-bottom: 20px;
        }

        #menu p {
        font-size: 11px;
        }
