Welcome to FS Compare (FSC)

FS Compare (FSC) is controlled through keyboard commands exclusively, listed below. Due to some bug in integrating the JDIC WebBrowser component, keyboard bindings are sometimes not working, so use the "Do" menu to move around.

To return to the demos, press ALT-N to advance to the next in sequence.

ActionKeystroke
Help screenALT-H
Increase font sizeCTRL-I
Reset font sizeCTRL-0 (zero)
Decrease font sizeCTRL-D
Expand screen sizeALT-L
Shrink screen sizeALT-S
Next demo in listALT-N
Choose demo from listALT-C
Reload pageALT-R
Reload list of files and display first in listALT-F
