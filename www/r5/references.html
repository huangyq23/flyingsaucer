CSS and XHTML References

Library

Flying Saucer makes use of the CSS Parser project on Sourceforge, formerly the Steady State CSS parser. All of our modifications to it have been contributed back to this project per the LGPL license.

The Specs:

  • the W3C XHTML spec
  • W3C CSS 2.1 spec (exhaustive but precise)
  • The W3CSchools XHTML site a more hands on definition of XHTML with examples and tutorials

Learning about good CSS design

  • A List Apart, years of articles about good CSS design
  • CSS Edge demos and cool tutorials of advanced CSS
  • CSS Zen Garden An example site showing the power of CSS.
  • Dashboard Apple's upcoming technology to free XHTML/CSS from the webbrowser.

Books

  • Eric Myers on CSS
  • CSS Pocket Reference, 2nd Edition (only 10$!)
